QuiltSoft Designer is a black and white (or green and black on QB Systems Machines) ASCII quilt design program. This entirely command line software renders quilt patterns in graphical ASCII text output and produces procedurally generated quilt patterns. A special miniature music haiku tracker program creates an 8bit audio ditty to complement each visual quilt design. In Town Mode, a 2d ASCII town is procedurally rendered and the user can traverse the town, visiting various inhabitants and their quilts within the rendered environment. These quilts and songs can be collected and stored for later retrieval.
